Какое значение будет в переменных b, с и d после выполнения данного кода.

int a = 120;
byte b = a + 10;
byte c = (byte)(a + 10);
byte d = a + 1;
Объясните полученный результат.

1. Инициализация переменной a

int a = 120;
Тип int (целое 32-битное число) инициализируется значением 120.

2. Инициализация переменной b

byte b = a + 10;
Разбор:
a имеет тип int, 10 — это литерал типа int.
При выполнении арифметической операции a + 10, результат также будет типа int (по умолчанию все целочисленные операции возвращают int).
Попытка присвоить значение типа int переменной типа byte без явного приведения вызывает ошибку компиляции.
Ошибка компиляции: Type mismatch: cannot convert from int to byte.
Причина ошибки:

Тип byte имеет размер 8 бит, диапазон значений от -128 до 127.
Результат выражения a + 10 равен 130, что не может быть неявно преобразовано из int в byte.
Что нужно сделать для исправления?

byte b = (byte) (a + 10);
3. Инициализация переменной c

byte c = (byte)(a + 10);
Разбор:
Как и в случае с b, результат a + 10 равен 130.
Явное приведение типа (byte)(a + 10) принудительно конвертирует int в byte.
При преобразовании из int в byte, если значение выходит за диапазон -128 до 127, происходит переполнение по модулю 256.
Значение 130 не входит в диапазон byte. Для получения результата используется метод взятия по модулю:
130 mod 256=130
Но так как диапазон byte — это от -128 до 127, то значение 130 трактуется следующим образом:
130 — это то же самое, что и 
(−128+2)=−126.
В двоичной системе 130 = 10000010 (в 8 битах), но для знакового числа это двоичное представление соответствует значению -126.
Значение c:

c = -126;
4. Инициализация переменной d

byte d = a + 1;
Разбор:
Операция a + 1 возвращает int со значением 121.
Неявное приведение из int в byte невозможно, это приводит к ошибке компиляции:
Ошибка компиляции: Type mismatch: cannot convert from int to byte.
Даже несмотря на то, что 121 находится в диапазоне byte (-128 до 127), автоматическое приведение не выполняется, так как арифметическая операция с int возвращает int.
